# 工作需求 · 匹配自检（模板）

> 整理自当前求职方向、MBTI、工作经验、价值观与技能。**用法**：有新的意向岗位时，用本文档逐项对照，看岗位与自身需求是否匹配。可复制到工作区后填写，或通过 Cursor JD 自检 skill 的「场景零」访谈生成内容。

---

## 一、如何使用本文档

1. 拿到一个 JD 或机会时，打开本文档，从 **第二节** 起逐块对照。
2. 在「**八、匹配自检清单**」里逐条打勾或写备注，看有多少项符合、哪些是硬伤。
3. 若**硬性约束**不满足（地点、弹性、薪资、节奏），优先排除或先问清再投。
4. 若**价值观与职业追求**与岗位方向相反，慎重。
5. **使用 Cursor 做新 JD 自检**：对 Cursor 说「**对这份 JD 做自检**」，并粘贴 JD 全文或给出工作考虑文档路径；Agent 会按本文档的技能点与清单自动比对并输出结论。

## 八、匹配自检清单（新岗位时逐条对照）

把意向岗位/JD 对照下面清单，符合打勾或写备注，便于决定是否投递或接 offer。

### MBTI 与意义感

- [ ] 工作内容是否有「对人/用户/社会」的可感知影响？
- [ ] 团队文化是否偏「用户价值与反馈为真、少冗长流程」？
- [ ] 团队规模/组织结构是否偏简单（人少、层级少）？
- [ ] 沟通方式是否高效直接、会议是否有效而非冗长？
- [ ] 是否有一定话语权与成长空间，而非纯执行？

### 经验与能力

- [ ] 岗位是否落在你的目标方向（如产品/体验/内容等）？
- [ ] 是否用到你已有的核心能力？
- [ ] 若有明显 gap，是否愿意且能在入职后补？

### 价值观与追求

- [ ] 是否做有杠杆、有规模的事（或你个人追求的方向）？
- [ ] 行业/产品是否与你兴趣或特长至少一类契合？

### 硬性约束

- [ ] 地点是否可接受？通勤/远程是否可兼顾？
- [ ] 弹性（到岗天数、早走/远程）是否问清且可接受？
- [ ] 薪资是否在可接受区间？
- [ ] 压力与节奏是否不会长期超过可承受范围？
